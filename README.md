# Ashickey Platform V2 🚀
A modern, aggressively optimized Full-Stack PWA encompassing advanced Telemetry, Background Queue Engines, and Multi-platform delivery frameworks.

## 📍 System Architecture
- **Web App (Frontend):** Flutter PWA natively injecting dynamic geographic locators and dynamic CanvasKit rendering out-pacing generic web-scrapers.
- **Microservices (Backend):** Laravel REST Engine isolated entirely behind an internal Docker Bridge. 
- **Database Vaults:** MongoDB 7 natively processing aggregated telemetry logs safely decoupled from global relational bottlenecks!
- **Data Broker:** Redis 7 powering high-fidelity asynchronous email blast distributions sequentially line-by-line avoiding host connection timeouts!

## 🕒 The Advanced Feature Matrix

### Dual-Factor Telemetry Classification
Analytics on Ashickey deliberately cut out "bounce statistics" by forcing organic algorithm checkpoints natively across standard views mapping strictly between 3 modes:
1. `app_open`: Standard OS-level system triggers the exact ms the PWA mounts.
2. `just_view`: Basic extraction verifying a user physically instantiated a Modal component.
3. `actual_reader`: Flutter physically anchors `Timer(15s)` natively merged dynamically alongside a `< 30%` physical Scroll Depth limit explicitly tracking dynamic reader ratios against the base!

### Timezone Standardization (Dashboard Visuals)
- **Deployment:** Standardized internal core environments directly around `APP_TIMEZONE=Asia/Kuala_Lumpur` structurally fixing generic `UTC` shift boundaries dynamically.
- **Graphical Visualizations:** Chart.js intrinsically bound natively across the Admin Portals actively monitoring explicit viewer behaviors!

## 🔒 Security & Deployment Ruleset

### 1. Web-Node Firewalls
If deploying onto Hostinger, **absolutely strictly** isolate outbound properties to ensure your Docker Bridge natively secures the Database ports from brute force vectors globally! You must strictly set 3 Accept rules:
- **Port 22 (TCP):** SSH Tunnel Access.
- **Port 80 (TCP):** Un-encrypted Web Traffic.
- **Port 443 (TCP):** Let's Encrypt SSL Protocols.

### 2. Frontend Source Protection
PWA engine automatically nullifies physical User-Agent DOM Scraping engines natively evaluating across:
- Blocked F12 (DevTools Hooking).
- Touch Selectors entirely disabled across the Web Framework structure to block content scraping algorithms passively!

## ⚙️ Initial Startup

### Local Environment
1. Ensure your `.env` contains:
```env
APP_TIMEZONE=Asia/Kuala_Lumpur
REDIS_HOST=redis
MAIL_ENCRYPTION=ssl
ADMIN_EMAIL=your-admin@gmail.com
```
2. Spawn the isolated structure natively:
```bash
docker compose up -d --build
```
3. Your secure private dashboard sits tightly coupled behind `https://ashickey.space/kcg5025/login`.
